Variable types
==============

.. image:: ../img/TWP05_015.png
   :height: 8.507cm
   :width: 16.595cm
   :align: center
   :alt: 

+ The content of a variable has a type.
+ The type defines the nature of the data that the variable stores.
+ The most common types are:
    + **Numeric types:** integers, floating point numbers.
    + **Strings or text:** can store numbers, letters and other characters.
    + **Boolean types:** store values as True and False.
