Python Tools
======================

+ Conditions: ``if``, ``elif``, ``else``
+ Loop: ``while``
+ Assignment operator: ``=``
+ Equality operator: ``==``
+ Inequality operator: ``!=``
+ Display a message: ``print()``
+ Read user input: ``input()``
+ Convert to integer: ``int()``
+ Generate a random integer: ``randint()``

These are some of the tools. There are many more!
